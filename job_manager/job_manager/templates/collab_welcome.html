Getting started with the HBP Neuromorphic Computing Platform

  1. Add team members to your Collab using the "Team" link in the navigation bar on the left. Only members of the team will be able to launch simulations. If this is a public collab, anyone with an HBP account will be able to view the results. If it is a private collab, only team members will be able to view results.
  2. Read the Guidebook.
  3. Request access to the platform, using the "Resource Manager" link.
  4. Run simulations, using the "Job Manager" link.

You can now remove or edit this "Overview" page, and replace it with documentation for your project.
